核心逻辑链条

宪法是“权力的限制书”（第三单元），也是“权利的保障书”（第二单元）。规范权力运行是保障人权的前提，保障权利实现是规范权力的目的。

学习重点提示

一切法律制度的运行最终都要归于对公平正义（第四单元）的坚守，这是法治国家的最高价值追求，也是宪法至上原则的终极落地。
