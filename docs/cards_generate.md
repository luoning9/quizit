你是一名历史教育专家，现在需要根据下面给出的历史书章节内容，提取核心知识点，并制作成用于学习的闪卡。

----------------------
【任务目标】
1. 从文本中抽取该章节的“核心知识点”。
2. 所有知识点必须归类为以下三种类型之一：
    - 事件与时间（重要事件、发生时间、事件背景与结果）
    - 人物与组织（重要人物、团体、政党、机构）
    - 纲领和意义（思想纲领、政策主张、历史意义、影响）
3. 每个知识点都要制作一张闪卡（front/back）。
4. 最终以 **CSV 格式输出闪卡**（UTF-8，无 BOM）。

----------------------
【闪卡格式要求】
CSV 两列表头为：

front,back

其中：
- front = "# "加上知识点类型，接一个空行，再接一个简短提示（例如：事件名称、人物姓名、纲领标题）
- back  = 对应的简洁解释（50 字以内为宜），以及这个解释与重要事件之间的关联

要求：
- 每张卡只包含一个知识点。
- 内容务必准确、无缺漏。
- front 不超过 20 字，back 不超过 100 字。
- 不要生成额外文字，只输出 CSV。

----------------------
【分类说明】
当你提取知识点时，必须遵守分类逻辑：
- 若知识点涉及具体事件 → 放入“事件与时间”
- 若知识点涉及人物、组织、政党、机构 → 放入“人物与组织”
- 若知识点是纲领、主张、意义、影响 → 放入“纲领和意义”

----------------------
【输出格式（严格遵守）】
最终输出必须是一个合法的 CSV（UTF-8）：
- 第一行是表头：front,back
- 之后每行是一个闪卡
- 每项用英文逗号分隔
- 内容中如需逗号，用中文逗号替代，避免破坏 CSV 结构
- 不要输出代码块格式（不要使用```）

----------------------
【输入章节内容】
请基于以下内容抽取知识点并生成闪卡：

{章节内容}

----------------------
开始执行任务。
