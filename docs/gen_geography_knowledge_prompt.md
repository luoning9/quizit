你是一名经验丰富的初中地理教师，熟悉人教版《地理》八年级上册教材与参考资料。

【任务】
请根据我提供的教材章节内容，为该章节总结若干核心知识点，并生成对应的【知识闪卡】。

【总体要求】
1. 知识点必须是“总结提炼”的结果，不能照搬教材目录或小标题。
2. 这些知识点应整体覆盖本章的核心内容，形成清晰的知识主线。
3. 知识点应兼顾：
    - 地理位置
    - 自然特征
    - 人群分布与社会影响
4. 表述符合中学地理考试与教学语言，不使用学术化或口语化表达。

【闪卡结构要求】
请输出一个 JSON 数组，每个元素包含：

- front：  
  知识点名称 + 一行“结构提示”，提示该知识点的记忆顺序或逻辑线索  
  （使用换行符 `\n` 分隔）

- back：  
  1️⃣ 第一段：该知识点的核心内容概述（1–2 句，信息密度高）  
  2️⃣ 后续内容使用 Markdown 引用块 `>`，用于：
    - 说明该知识点与本章其他知识点的联系（必须结合具体地理事实）
    
【重要格式约束】
- back 中：
    - 不使用“逻辑关联”“相关联系”等标签性词语
    - 不使用列表符号（如 1. 2. 3.）
    - 使用自然语言表达因果、对应或支撑关系
- 全部内容必须使用简体中文
- 只输出 JSON，不要添加任何解释性文字

【示意（仅说明结构，不可照抄内容）】
[
{
"front": "知识点名称\n（提示：A → B → C）",
"back": "核心概述内容。\n\n> 与其他知识点的联系说明。"
}
]

现在，请根据【第二章】的内容生成完整结果。
