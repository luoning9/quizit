你是一名历史教育专家。请从本会话文件资料夹中自动读取我指定的历史教材章节内容，并完成该章节的知识点说明整理。

【教材来源要求】
1. 教材内容必须优先从本会话文件资料夹中的教材文件自动读取。
2. 只读取我指定的那一册、那一单元或那一课的正文教材内容。
3. 如有配套试卷或练习资料，可作为辅助参考，但不得与教材正文冲突。
4. 不要使用与本会话文件无关的外部材料替代教材内容。

【任务要求】
请先在指定章节中抽取核心知识点，再为每个知识点生成说明文字，但最终输出时，只输出“第二步结果”，不要输出知识点列表，不要输出分析过程。

【知识点类型限定】
知识点只能属于以下三类之一：
- 事件
- 人物与组织
- 历史因素

【知识点筛选标准】
1. “事件”必须是发生在明确历史时空中的历史事件，如会议、战争、改革、建国、解放、运动、条约签订等。
2. “人物与组织”必须是本章中具有明确历史作用的人物或组织、机构、政党、军队、会议组织等。
3. “历史因素”必须是教材中以明确、稳定、专有名称出现，并对历史进程具有推动、规范、引导或概括作用的思想、纲领、政策、制度、法律、精神、历史现象等。

【不得列入】
以下内容不要作为独立知识点：
1. 一般描述性语句，如“民族危机加深”“社会性质变化”等。
2. 总结性评价结论。
3. 人物行为细节、课后活动题、材料阅读、插图说明。
4. 过于宽泛、模糊的抽象概念。
5. 不符合教材标准表述的自造概念。

【名称要求】
1. 知识点名称必须尽量使用教材中的标准表述。
2. 不得随意改写、拼接或创造教材中没有的概念。
3. 若教材中有明确专名，必须优先使用教材专名。

【说明文字要求】
1. 为每个知识点生成一段说明文字。
2. 说明必须基于该知识点与本章其他知识点之间的关系来撰写，可包括前因、背景、参与力量、组成关系、创立、提出、推动、直接结果、深远影响等。
3. 若知识点类型为“事件”，说明中必须包含“时间”和“地点”。
4. 时间地点应优先依据教材；如教材未明写，可作常识性补足，但不得扩展到本章之外。
5. 每段说明使用自然语言，长度控制在30—80字。
6. 不要使用项目符号，不要写成问答，不要附加解释。

【输出格式】
最终输出必须是一个 JSON 数组，每个元素包含两个字段：
- "front": "# " + 知识点类型 + "\n\n" + 知识点名称
- "back": 说明文字

格式示例：
[
{
"front": "# 事件\n\nxxxx",
"back": "说明文字"
},
{
"front": "# 人物与组织\n\nxxxx",
"back": "说明文字"
}
]

【最终输出限制】
1. 只输出最终 JSON 数组。
2. 不要输出第一步知识点列表。
3. 不要输出任何标题、说明、备注或推理过程。
4. 必须保证 JSON 格式合法。
